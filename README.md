Sales Data Exploratory Analysis using Python, Pandas, Matplotlib and Seaborn

## Project 1 — Sales EDA (Exploratory Data Analysis) <a name="project-1"></a>

**Folder:** `sales-eda-python/`  
**File:** `sales_eda.py`

### What it does:
Full exploratory analysis of a sales dataset with 12 business insights and visualizations:
1. KPI Overview (Revenue, Profit, Margin, Orders)
2. Monthly Revenue & Profit Trend
3. Revenue by Region (bar + pie)
4. Category Revenue vs Margin (bubble chart)
5. Customer Type Comparison
6. Profit Margin Distribution
7. Discount Impact Analysis
8. Correlation Heatmap
9. Quarterly YoY Comparison
10. Day-of-Week Order Patterns
11. Top Sales Representatives
12. Pareto 80/20 Analysis

### How to run:
```bash
pip install pandas numpy matplotlib seaborn
python sales_eda.py
# Charts saved to: charts/ folder
```

### Key Findings:
- Electronics (Laptops + Mobiles) drive ~38% of revenue
- Corporate customers generate 2.5x higher average order value than Retail
- Discounts > 10% reduce profit margins by ~8 percentage points
- Q4 shows consistent 20%+ revenue spike vs Q2
